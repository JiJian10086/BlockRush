# BlockRush

# ðŸ›¡ï¸ Block Rush: A Conceptual Tower Defense Game

**A conceptual Tower Defense web game built on the KiRo platform using Natural Language Interaction.**

## ðŸš€ Project Overview

"Block Rush" is an innovative web-based Tower Defense (TD) game that cleverly integrates **Blockchain security and mechanism concepts** into its core gameplay.

Players must utilize various **Blockchain Tools** (such as L2 Channels, ZK Snipers, Smart Contracts) to build their defenses while achieving a dual objective:
1. **Defense (Block):** Repel various **Malicious Attacks** (e.g., 51% Attack, Double Spending).
2. **Passage (Rush):** Ensure **Legitimate Transactions** safely pass through the defense line.

This project aims to explore the potential of gamified education and demonstrate the efficiency of **Natural Language Driven Development**.

## ðŸ’¡ Technical Highlights and Innovation

### 1. Conceptual Mechanism Design
We translated complex blockchain concepts into game mechanics:
* **Defense Towers:** Mapped to real-world technical tools. E.g., `L2 Channel` (Layer2 Scaling) provides sustained damage and health suppression; `Mainnet Cannon` (Hard Fork) deals percentage damage against BOSS units.
* **Enemies:** Mapped to network threats. E.g., `Double Spending` is an invisible unit; `51% Attack` is a high-HP, high-damage reduction BOSS.

### 2. Natural Language (NL) Driven Logic Implementation
The core logic implementation departs from traditional coding, utilizing the **Natural Language (NL) Interaction** feature of the KiRo platform for definition and optimization:
* Logic is defined by "chatting" with KiRo to specify tower attributes, enemy behavior patterns, and complex multi-stage logic.
* This method enables **incremental, document-driven** logic optimization, effectively reducing bugs and accelerating prototype iteration speed.

### 3. Engineering Status
* The Demo is initially playable.
* The project code is fully deployed and open-sourced within the **ModelScope Studio**.
